AI-Driven Inventory Intelligence System
An advanced automation ecosystem that transforms raw warehouse data into actionable business insights using AI and low-code tools.

🌟 Key Features:
Real-time Currency Sync: Automatically fetches current EUR/UAH exchange rates via HTTP API.

Automated Data Processing: A custom Python/JS logic layer processes inventory balances, calculates revenue, and identifies top-performing products.

AI Business Insights: Powered by Google Gemini, the system generates a professional weekly summary and provides smart restocking recommendations.

Multi-channel Reporting: Synchronizes final analytics with Notion databases and sends stylized HTML reports via Gmail.

🛠 Tech Stack:
Automation: n8n

Intelligence: Google Gemini AI

Data: Python, Pandas, Excel (XlsxWriter)

Integrations: Notion API, Gmail API

Интеллектуальная система складской аналитики на базе ИИ
Продвинутая экосистема автоматизации, которая превращает «сырые» складские данные в ценные бизнес-инсайты с помощью ИИ и инструментов low-code.

🌟 Основные возможности:
Синхронизация валют: Автоматическое получение актуального курса EUR/UAH через API в режиме реального времени.

Автоматизированная обработка данных: Программная логика (Python/JS) обрабатывает остатки на складе, рассчитывает выручку и выявляет наиболее продаваемые товары.

Бизнес-аналитика с ИИ: Благодаря Google Gemini, система формирует профессиональное резюме недели и дает рекомендации по закупкам.

Многоканальная отчетность: Результаты синхронизируются с базой данных Notion и отправляются в виде стилизованных HTML-отчетов через Gmail.

🛠 Технологический стек:
Автоматизация: n8n

Интеллект: Google Gemini AI

Данные: Python, Pandas, Excel (XlsxWriter)

Интеграции: Notion API, Gmail API
